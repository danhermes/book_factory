# Book Factory Test Suite

This directory contains test scripts and utilities for the Book Factory CLI.

## Quick Start

1. Install dependencies:
   ```bash
   pip install -r test_requirements.txt
   ```

2. Run a quick test:
   ```bash
   python quick_test.py outline
   ```

3. Run all tests:
   ```bash
   python run_all_tests.py
   ```

## Available Scripts

- `test_book_cli.py` - Main test script
- `test_book_cli.sh` - Shell script for Linux/macOS
- `test_book_cli.bat` - Batch script for Windows
- `quick_test.py` - Quick test script
- `run_all_tests.py` - Script to run all tests
- `programmatic_book_cli_example.py` - Example of programmatic usage
- `analyze_test_results.py` - Script to analyze test results
